# Customer Segmentation Analysis

This project performs customer segmentation for an e-commerce dataset.

## Objectives
- Analyze customer purchase behavior
- Perform data visualization
- Apply K-Means clustering
- Identify different customer segments

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Retail Sales Dataset

## Output
Customers were grouped into three segments based on:
- Age
- Quantity Purchased
- Total Spending
