# Customer Segmentation Analysis

## Project Overview
This project performs customer segmentation using marketing analytics data. 
The goal is to analyze customer purchasing behavior and group customers into different segments using clustering techniques.

## Dataset
The dataset contains information about customers including:
- Age
- Income
- Product spending
- Purchase history

Dataset: marketing_campaign.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps in the Project
1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Data Visualization
5. Customer Segmentation using K-Means Clustering
6. Cluster Analysis

## Key Insights
Customers were segmented based on their income and total spending behavior. 
This helps businesses understand different customer groups and design targeted marketing strategies.

## Output
The project generates visualizations including:
- Income vs Spending plot
- Age distribution
- Elbow method graph
- Customer segmentation clusters
