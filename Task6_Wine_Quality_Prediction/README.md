Wine Quality Prediction
Overview

This project predicts wine quality based on its chemical properties using machine learning techniques. The dataset contains features such as acidity, density, pH, sulphates, and alcohol which influence the quality of wine.

Three classification models are implemented to predict wine quality:

Random Forest Classifier

Stochastic Gradient Descent (SGD) Classifier

Support Vector Classifier (SVC)

Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Scikit-learn

Output

The models are trained and evaluated to predict whether a wine is good or bad quality, and their performance is compared using accuracy scores and visualizations.
