House Price Prediction using Linear Regression
Project Overview

This project focuses on predicting house prices using a Linear Regression Machine Learning model. The model analyzes different features from a housing dataset and estimates the price of a house based on those inputs.

Linear Regression is one of the most basic and widely used algorithms in machine learning for predicting continuous numerical values. In this project, we implement the full machine learning workflow including data preprocessing, model training, evaluation, and visualization.

This project was developed as part of the Oasis Infobyte Internship Program.

Dataset

The dataset contains various features related to houses such as:

Area

Number of Bedrooms

Number of Bathrooms

Number of Stories

Parking Availability

Air Conditioning

Basement

Furnishing Status

Preferred Area

Main Road Access

Target Variable:

House Price

Technologies Used

Python

Pandas

NumPy

Matplotlib

Scikit-learn

Project Workflow
1. Data Collection

The housing dataset is loaded and analyzed to understand the available features.

2. Data Preprocessing

Categorical variables are converted into numerical format and unnecessary data issues are handled.

3. Feature Selection

Relevant features are selected to train the prediction model.

4. Model Training

A Linear Regression model is trained using the Scikit-learn library.

5. Model Evaluation

The model performance is evaluated using:

Mean Squared Error (MSE)

R² Score

6. Visualization

A scatter plot is created to compare Actual House Prices vs Predicted Prices.

Output

The trained machine learning model predicts house prices based on input features and provides a visualization that shows how closely the predicted values match the actual values.**
