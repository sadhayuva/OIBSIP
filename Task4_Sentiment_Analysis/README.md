Sentiment Analysis
Project Overview

This project analyzes the Google Play Store Apps dataset to understand trends in app categories, ratings, installs, and pricing using Python.

Dataset

The dataset includes information such as App Name, Category, Rating, Reviews, Installs, Type (Free/Paid), Price, and Genres.

Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Key Analysis

Top app categories

Rating distribution

Most installed apps

Free vs paid apps comparison

Output

The project generates visualizations showing app category popularity, rating distribution, installation trends, and comparison between free and paid apps.
